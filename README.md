# Not Pacman On Arduino

### Pacman inspired game running on arduino and using LCD1602

Pacman inspired game using :

* Arduino uno
* LCD1602
  * rs      :   digital pin 7
  * enable  :   digital pin 6
  * d4      :   digital pin 5
  * d5      :   digital pin 4
  * d6      :   digital pin 3
  * d7      :   digital pin 2
* Potentiometer
  * to control arduino brightness
* Joystick Module
  * y       :   analog pin 0
  * x       :   analog pin 1
  * sw      :   digital pin 8
* Buzzer
  * pin     :   digital pin 9

## How to compile and upload

Either

```bash
make upload
```

or

open using arduino IDE
